# Solidjs-Solidity-Starter🔥
## 👋Summary
SolidJS/Solidity/tailwindCSS/Truffle/Ganache-sliを使用したモノレポです
## 🎉Usage
```bash
$ yarn install
```
## 🛠️Add/Remove/Update package
```bash
$ yarn client add/remove/update {PACKAGE_NAME}
$ yarn contract add/remove/update {PACKAGE_NAME}
```
